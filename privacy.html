개인정보 처리방침

최종 수정일: 2025년 11월 16일 | 시행일: 2025년 11월 16일

1. 개요

PlayPing (플레이핑) (이하 "앱") 사용자의 개인정보를 중요하게 생각하며, 개인정보보호법 등 관련 법령을 준수하고 있습니다.

이 개인정보 처리방침은 앱을 통해 수집되는 개인정보의 종류, 수집 목적, 이용 방법, 보관 기간, 보호 조치 등에 대해 설명합니다.

2. 수집하는 개인정보 항목

2.1 필수 정보

계정 정보

  • 이메일 주소 (이메일 로그인 사용 시)
  • 사용자 고유 식별자 (UID)
  • 닉네임/표시 이름

게임 활동 정보

  • 게임 플레이 기록
  • 게임 점수 및 순위
  • 게임방 생성/참가 기록

2.2 자동 수집 정보

기기 정보

  • 기기 유형 및 모델
  • 운영 체제 및 버전
  • 앱 버전

광고 관련 정보 (Google AdMob 사용)

  • 광고 식별자 (IDFA/GAID)
  • 광고 상호작용 데이터
  • 광고 노출 정보

3. 개인정보 이용 목적

  • 서비스 제공: 사용자 인증, 게임 기록 저장, 실시간 멀티플레이어 기능
  • 서비스 개선: 앱 성능 분석, 버그 수정, 사용자 경험 개선
  • 광고 제공: 맞춤형 광고 표시, 광고 성과 측정
  • 커뮤니케이션: 서비스 관련 공지사항, 문의사항 응대

4. 개인정보 제3자 제공

Google Firebase

  • 제공 정보: 이메일, UID, 게임 데이터
  • 목적: 사용자 인증, 데이터 저장 및 동기화
  • 개인정보 처리방침: https://firebase.google.com/support/privacy

Google AdMob

  • 제공 정보: 광고 ID, 기기 정보
  • 목적: 광고 표시 및 성과 측정
  • 개인정보 처리방침: https://policies.google.com/privacy

5. 개인정보 보관 기간

  • 계정 정보: 회원 탈퇴 시까지
  • 게임 데이터: 서비스 이용 기간 동안, 계정 삭제 시 30일 내 삭제
  • 로그 데이터: 수집일로부터 1년

6. 사용자의 권리

  • 정보 접근권: 수집된 개인정보 확인 요청
  • 정보 수정권: 부정확한 정보 수정 요청
  • 정보 삭제권: 계정 삭제 요청 (앱 내 설정 > 계정 삭제)
  • 광고 추적 거부권: 기기 설정에서 광고 추적 제한

7. 개인정보 보호 조치

  • 데이터 암호화: 전송 중 TLS/SSL 암호화
  • 접근 제어: 인증된 사용자만 데이터 접근
  • 보안 서비스: Firebase Security Rules 적용
  • 정기 보안 점검: 취약점 모니터링

8. 아동의 개인정보

앱은 모든 연령대를 대상으로 하지만, 14세 미만 아동의 경우 법정대리인의 동의가 필요할 수 있습니다.

9. 개인정보 처리방침 변경

변경 사항은 앱 내 공지 또는 이메일로 통지하며, 중요 변경 사항은 최소 7일 전 사전 고지합니다.
